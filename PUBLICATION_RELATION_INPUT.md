# Strong Related-Publication Ties

Use this file as a scratchpad for manually specifying especially strong bidirectional related-publication ties.

## Format

Send ties as number pairs, one per line:

```text
5-15
5-7
20-11
```

Optional notes are fine:

```text
5-15  # DSL/LLM annotation pair
20-11 # embedding inference pair
```

I will translate these temporary numbers into stable publication slugs in a site data file. The resulting behavior should be: manual ties are bidirectional, forced into Related Publications when either paper is viewed, and sorted above ordinary tag/coauthor recommendations.
